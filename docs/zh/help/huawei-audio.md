# 华为荣耀面对面设置

:::warning
华为荣耀手机插入voicepie耳机设备，面对面模式下，输出音量设备不能直接切换，需手动自行切换

手机屏顶部下拉控制中心-打开音频控制-选择本机-即可外放
:::
