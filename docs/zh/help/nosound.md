# 使用翻译时没有声音

:::tip

翻译没有声音存在很多可能的原因，可以参考以下解决办法去一一排解

:::

## 播报设置

在使用翻译时，播报设置请勾选对方/我方 播放翻译

## 媒体音量调节

耳机设备播放属于媒体音量输出，在进行实体通话时，需要提高媒体音量
