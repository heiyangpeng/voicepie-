:::tip 如何使用 帮助与支持？

当您遇到使用问题时，请根据以下问题进行自行排解，也是为了减少彼此沟通成本，望您理解。

如需要获得一对一服务支持，请留下您的联系方式和问题，在<code>反馈问题</code>中提交。

因人力支持有限，我们会尽快与您联系!

:::

:::warning 注意事项
若有升级更新，请升级到最新APP版本，旧版本可能存在遗留bug。

安装APP时同意并给予所有权限，请勿跳过，若拒绝或跳过请关掉APP后台，重新打开给予权限。
:::
