# Unable to Connect to Server

::: warning App shows: Cannot connect to server
:::

## Open Internet Options

Click Windows icon and select <code>Internet Options</code>

![](https://bu.dusays.com/2025/02/07/67a586f2331f0.jpg)

## Click <code>Connections</code>

![](https://bu.dusays.com/2025/02/07/67a586f169a3c.jpg)

## Click <code>LAN Settings</code>

![](https://bu.dusays.com/2025/02/07/67a586f16fadf.jpg)

## Ensure All Options are Unchecked

![](https://bu.dusays.com/2025/02/07/67a586f169d8a.jpg)

## Click <code>OK</code>

- Click <code>Apply</code> (Cannot skip)
- Click <code>OK</code> again

![](https://bu.dusays.com/2025/02/07/67a586f16d094.jpg)
